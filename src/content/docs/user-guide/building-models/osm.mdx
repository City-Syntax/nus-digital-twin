A urban-scale model provided by OpenStreetMap is available on the platform. 

<Steps>
1. Select the <span style={{ display: 'inline-block' }}><Icons.OSMBuildings height="1.125rem" width="1.125rem" /></span> OSM Buildings button from menu on the left. If your menu appears on the bottom, select the <span style={{ display: 'inline-block' }}><Icons.Layers height="1.125rem" width="1.125rem" /></span> button to find the OSM Buildings button.
2. Select On to turn on the OpenStreetMap urban-scale model.
</Steps>

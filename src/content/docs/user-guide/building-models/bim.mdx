BIM models are available for certain buildings in NUS. To switch to BIM models:

<Steps>
1. Select the <span style={{ display: 'inline-block' }}><Icons.BIMModels height="1.125rem" width="1.125rem" /></span> BIM Models button from menu on the left. If your menu appears on the bottom, select the <span style={{ display: 'inline-block' }}><Icons.Layers height="1.125rem" width="1.125rem" /></span> button to find the BIM Models button.
2. Select On to turn on the BIM models.
</Steps>

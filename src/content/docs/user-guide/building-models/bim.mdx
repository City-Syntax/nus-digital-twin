Building information models (BIM) are available for certain buildings in NUS. To switch to BIM:

<Steps>

1. Select the <span style={{ display: 'inline-block' }}><Icons.BIM height="1.125rem" width="1.125rem" /></span>
   BIM button from menu on the left. If your menu appears on the bottom, select the <span style={{ display: 'inline-block' }}><Icons.Layers height="1.125rem" width="1.125rem" /></span> button to find the BIM button.

2. Select On to turn on BIM.

</Steps>

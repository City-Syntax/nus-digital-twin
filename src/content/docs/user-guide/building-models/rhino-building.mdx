Building-level Rhino models are available for some buildings in NUS.

<Steps>
1. Select the <span style={{ display: 'inline-block' }}><Icons.RhinoModels height="1.125rem" width="1.125rem" /></span> Rhino (Building) button from menu on the left. If your menu appears on the bottom, select the <span style={{ display: 'inline-block' }}><Icons.Layers height="1.125rem" width="1.125rem" /></span> button to find the Rhino (Urban) button.
2. Select On to turn on Rhino building-level models.
</Steps>

:::note
A [urban-scale rhino model](../rhino-urban) is also available on the platform.
:::

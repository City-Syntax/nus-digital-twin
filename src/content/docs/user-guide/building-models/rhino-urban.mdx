A Rhino urban-scale model is available on the platform. To display the Rhino urban-scale model:

<Steps>
1. Select the <span style={{ display: 'inline-block' }}><Icons.RhinoModels height="1.125rem" width="1.125rem" /></span> Rhino (Urban) button from menu on the left. If your menu appears on the bottom, select the <span style={{ display: 'inline-block' }}><Icons.Layers height="1.125rem" width="1.125rem" /></span> button to find the Rhino (Urban) button.
2. Select On to turn on the Rhino urban-scale model.
</Steps>

:::note
A set of [building-level rhino models](/user-guide/building-models/rhino-building) are also available on the platform.
:::

This article explains how to use the Distance controls available on the platform. The distance control enables you to color buildings starting from a selected central building. The default central building is the Singapore Synchrotron Light Source.

## Enabling Color By Distance

<Steps>
1. Click on the <span style={{ display: 'inline-block' }}><Icons.Distance height="1.125rem" width="1.125rem" /></span> Distance button on the menu on the right. If your menu appears on the bottom, select the <span style={{ display: 'inline-block' }}><Icons.Controls height="1.125rem" width="1.125rem" /></span> button to locate the Distance button.
2. Select On to enable color by distance.
</Steps>

## Select a New Central Building

<Steps>
1. Click on the <span style={{ display: 'inline-block' }}><Icons.Distance height="1.125rem" width="1.125rem" /></span> Distance button on the menu on the right. If your menu appears on the bottom, select the <span style={{ display: 'inline-block' }}><Icons.Controls height="1.125rem" width="1.125rem" /></span> button to locate the Distance button.
2. Click on 'Select Central Building' button.
3. Select a building on the map to set it as the new central building.
</Steps>

NUS Digital Twin offers a public API for its building data, found at [nus-digital-twin.com/api/v1/buildings](/api/v1/buildings).

Looking to learn how to use the API? Go to the [API Reference](/api-reference).

## Update Building API

The building API serves a subset of the data available in the `src/content/buildngs/buildings.json` file. To update what is served to users, look at the `src/pages/api/v1/buildings.ts` file.

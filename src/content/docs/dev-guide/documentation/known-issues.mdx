There is currently one known issue with the documentation site.

## Pagefind with Vercel Deployment

Due to an issue with the official [`@astrojs/vercel` adapter](https://github.com/withastro/adapters/tree/main/packages/vercel/) on Astro v5, pagefind is disabled on the existing deployment.

See the relevant GitHub issues:

- https://github.com/withastro/astro/issues/12663
- https://github.com/withastro/adapters/issues/445
